Project Description

Overview: A banking application using MongoDB, Node.js, Express, and React.
Features:
User Auth: Register and log in.
Account Management: View and update user info.
Transactions: Deposit, withdraw, and transfer funds.
Technical Details:
RESTful API: JSON-based routes.
Session Management: User session tracking.
Testing: API tested with Postman.
Challenges: My main difficulty was to understand how Boostrap worked and create a professional-looking website.

Languages/Framework Used

JavaScript (Node.js), React, MongoDB
