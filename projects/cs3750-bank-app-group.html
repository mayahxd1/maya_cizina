Project Description

Overview:
Developed a banking application with a comprehensive frontend and backend using the MERN stack (MongoDB, Express, React, Node.js). The app includes role-based access and banking functionalities.
Features:
User Authentication:
Secure login system with SHA-256 hashed passwords. Role-based access for administrators, employees, and customers.
Role-Specific Features:
Administrators: Full access to all screens, manage roles, view and manage accounts.
Customers: Manage savings, checking, and investment accounts, view transaction histories.
Employees: View and manage customer accounts, transfer funds between customers.
Account Management:
Deposit, withdraw, and transfer funds. Full and individual account transaction histories.
Technical Details:
Frontend: Clean, responsive UI using React. Backend: Express and MongoDB with RESTful APIs. Session Management: Secure user sessions.
Collaboration:
Version Control: Shared Git repository for collaborative development.

My role: Developed features for the customer role, including account management and transaction history.
Challenges:
The main challenge for me was to make my share of the project work with the rest of the group's code.

Languages/Framework Used

JavaScript (Node.js), React, MongoDB, Express
