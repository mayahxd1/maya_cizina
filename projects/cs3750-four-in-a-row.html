Project Description

Overview:
To demonstrate proficiency in React by implementing a four in a row game logic and creating an interactive user interface. Features:
Tile Dropping: Player can click on columns to drop their tiles into the lowest available spot.
Win Detection: Automatically detects four in a row horizontally, vertically, and diagonally.
Game End Conditions: Displays the winner and prevents further moves. Also detects and handles tie games.
Player Switching: Alternates turns between two players after each move.
Visuals: Colored tiles to distinguish between players.
Challenges: The main challenge for me was to understand how to implement the game logic and how to make the frontend using React.

Languages/Framework Used

React
